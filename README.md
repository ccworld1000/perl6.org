##### Perl 5 && Perl 6

> ​	讲到Perl 6 就不得不提Perl 5,后续我简写为P6 与 P5.
>
> 建议把P6当成一门新语言,可能对接触过P5的童鞋来讲是一样的.
>
> 不过他们确实很多相识点,更多细节需要通过学习深入了解,
>
> 祝大家在P6的路途愉快.
